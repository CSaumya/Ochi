# 🚀 Ochi UI Clone

A modern **UI clone of Ochi.design**, focused on smooth animations and a clean, premium user experience.

* 🎬 Framer Motion
* 🧩 GSAP

* Smooth scroll animations
* Section reveal effects
* Responsive design
* Interactive UI
